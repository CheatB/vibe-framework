# GTM (Go-to-Market)

Скилл для формирования стратегии вывода продукта на рынок: позиционирование, конкурентный анализ, каналы, лендинг, метрики запуска.

Основана на проверенной методологии — подробности в `references/methodology.md`.

## Когда использовать

- Фаза 1.8.3 в `/new-project` pipeline
- Standalone через `/business-analysis`
- При запуске нового продукта или пивоте

## Ключевые принципы

1. **Конкуренция = доказательство рынка.** Уникальная идея = нет рынка = нет денег
2. **Зелёный океан** — много мелких игроков, высокая частота, лёгкое переключение → лучший рынок для входа
3. **Средний чек**: B2C от $10, B2B от $150. Низкий чек + низкая частота = плохая идея
4. **80% времени на обратную связь, 20% на продукт**
5. **Правило 4 итераций**: ни один проект не провалился после 4 грамотных итераций
6. **Продавай до создания продукта** — landing + трафик + разговоры

## Workflow

```
1. Получить контекст → описание продукта, аватар, боль
2. Задать уточняющие вопросы по методологии
3. Провести ресёрч конкурентов:
   a. Спросить пользователя (знает ли конкурентов)
   b. Дополнить веб-поиском
   c. Показать результаты для валидации
4. Определить тип рынка (красный/синий/зелёный)
5. Выбрать стратегию отстройки
6. Рекомендовать каналы привлечения
7. Сформировать структуру лендинга
8. Определить метрики запуска
9. Записать результат в docs/business-analysis.md → секция "GTM"
```

## Уточняющие вопросы

1. Кто твой аватар? (пол, возраст, профессия, доход, контекст использования)
2. Какую боль решаешь? (конкретная ситуация, не абстракция)
3. Как решение работает? (пошагово, простым языком)
4. В чём количественная выгода? (сэкономит X часов, заработает Y рублей)
5. Знаешь конкурентов? (названия, ссылки — если не знаешь, найдём)

## Типы рынков

| Тип | Признаки | Стратегия |
|---|---|---|
| 🔴 **Красный океан** | Крупные игроки, высокие бюджеты, сложно конкурировать | Узкая ниша или уходить |
| 🔵 **Синий океан** | Нет конкурентов (или 1-2) | Рискованно: нет рынка? Или валидировать спрос |
| 🟢 **Зелёный океан** | Много мелких игроков, никто не доминирует, высокая частота | ✅ Лучший для входа |

## Стратегии отстройки

1. **Копирование 1:1** → другой регион / канал / аватар (тот же продукт, новый рынок)
2. **Адаптация** → другой подход к той же проблеме (бот вместо веб, AI вместо ручного)
3. **Без отстройки** → если рынок зелёный, можно конкурировать в лоб (лучше UX, цена, сервис)

## Каналы привлечения (decision tree)

```
Продукт ищут в поиске?
├── Да → Поисковый трафик (Google/Яндекс)
│   ├── Тёплые лиды, быстрые решения
│   └── Высокий CPC если конкурентная ниша
├── Нет → Социальные сети (Meta, TikTok, VK)
│   ├── Холодные лиды, нужна сильная фильтрация
│   └── Креативы + лендинг критичны
└── Оба → Комбо (поиск + соцсети)

Бюджет?
├── Есть → Платный трафик + органика
├── Минимальный → Outreach + органика
└── Нет → Только органика (долго, но бесплатно)

Telegram-бот?
├── Telegram Ads + посевы в каналах + органика
└── + Яндекс.Директ на лендинг → бот
```

## Структура лендинга (из методологии)

```
1. Заголовок (5%) — продукт или главная выгода + CTA
   Пример: "Подготовься к собесу за 2 недели"

2. Боль и ситуация (15%) — контекст, проблема, последствия
   Пример: "90% кандидатов проваливают case-раунд"

3. Выгоды и продукт (60%) — главная выгода + как работает пошагово
   Пример: "Бот с ежедневными задачами + разбор от менторов"

4. Оффер и CTA (20%) — что получит + что произойдёт после клика
   Пример: "Начать бесплатно в Telegram → [Кнопка]"
```

## Пример вывода

```markdown
## GTM (Go-to-Market)

### Аватар
Мужчина 25-35, менеджер в IT, доход 150-300К.
Ситуация: готовится к собесу, нужна структурированная подготовка.

### Конкуренты (результаты ресёрча)
| Конкурент | Цена | Аватар | Канал | Сильные стороны |
|-----------|------|--------|-------|-----------------|
| LeetCode | $35/мес | Разработчики | Поиск + органика | Огромная база задач |
| Practicum | от 80К | Начинающие | Контекст + блог | Менторство |

### Тип рынка: 🟢 Зелёный
- Много мелких игроков, нет доминирования
- Высокая частота (подготовка к собесу — регулярная потребность)

### Стратегия отстройки: Адаптация
- Узкий аватар (только менеджеры, не разработчики)
- Другой формат (бот в Telegram, а не веб-платформа)

### Каналы запуска
1. **Telegram Ads** (основной) — таргет на IT-каналы
2. **Посевы в каналах** — @managersunite, @productstar
3. **SEO лендинг** (долгосрок) — "подготовка к собесу менеджера"

### Структура лендинга
- Заголовок: "Подготовься к собесу за 2 недели"
- Боль: "90% кандидатов проваливают case-раунд"
- Продукт: Бот с ежедневными задачами + разбор
- CTA: "Начать бесплатно в Telegram"

### Метрики запуска (первый месяц)
- Цель: 500 пользователей бота
- CAC target: <300₽
- Конверсия лендинг→бот: >20%
- Retention D7: >40%
```

## Приоритет при конфликтах

1. Данные > мнение (если есть статистика — опирайся на неё)
2. Если конкуренты не найдены → это НЕ «нет конкурентов», а «плохо искали» или «нет рынка»
3. Зелёный океан > Синий океан (предпочитай проверенный рынок)
4. Быстрый запуск > идеальный продукт (landing + трафик + feedback → итерация)
5. Подробная методология → `references/methodology.md`
