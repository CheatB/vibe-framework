# Telegram Post Style — Стиль Ивана

Скилл для написания постов в узнаваемом стиле канала "Не просто Чел".

## Ключевые характеристики стиля

### Tone of Voice
- **Живой разговорный язык** — как будто беседуешь с другом за пивом
- **Ирония и самоирония** — не боимся посмеяться над собой
- **Откровенность** — делимся ошибками и факапами без прикрас
- **Мат и сленг** — уместно и органично (хер, пиздец, обсёр)
- **Эмпатия** — понимаем, что читатели в похожих ситуациях

### Структура постов

**Заголовок:**
```
***Заголовок через звёздочки и жирный
```
Примеры: "Путь из тысячи ошибок", "Чтобы что?", "Обезьяна и рутина"

**Тело поста:**

🔵 **Эмодзи-буллеты** для структуры:
- 🔵 — основные пункты
- 🟡 — дополнительные пункты или выводы

**Зачеркивания** для юмора:
```
<s>старых и красных</s> опытных и умелых
<s>смотреть рилсы</s> делать что-то понятное
```

**Вопросы к аудитории:**
```
// В конце поста через слэши
// Вас когда-нибудь сокращали?
// А вы вообще заморачиваетесь над такими вещали?
```

### Темы и подача

**Основные темы:**
- IT-менеджмент, аутсорс, работа
- Отцовство и семья
- Самооценка и ментальное здоровье
- Предпринимательство и ошибки
- Личная эффективность

**Подача:**
1. Начинаем с личной истории или наблюдения
2. Рефлексируем — почему так получилось
3. Делаем выводы или делимся инсайтами
4. Заканчиваем вопросом к аудитории

### Языковые паттерны

**Характерные фразы:**
- "Чет я думаю, что..."
- "Вот я думаю..."
- "Кажется, пора..."
- "Хер его знает"
- "Камон!"

**Сарказм и ирония:**
- "Мысль пиздец какая мудрая"
- "Весьма не вовремя. Хотя, когда это бывает 'вовремя'?"
- "успешных успехов"

### Форматирование

**HTML теги (для Telegram):**
- `<strong>Жирный текст</strong>`
- `<s>Зачеркнутый</s>`
- `<em>Курсив для вопросов</em>`
- `<br><br>` — двойной перевод строки между абзацами

**Эмодзи:**
- 🔵 🟡 — буллеты
- 🗿 — для вступления/о себе
- 🧠 — про инсайты

## Анти-паттерны

❌ **Не используй:**
- Официальный деловой язык
- Слишком правильную грамматику
- Излишнюю вежливость
- Абстрактные советы без личного опыта

✅ **Вместо этого:**
- Говори как с другом
- Будь честным про свои косяки
- Делись личным опытом
- Признавай, что сам не всё знаешь
