# Примеры постов в стиле автора

## Пример 1: Пост про ошибки и рефлексию

```
***Путь из тысячи ошибок начинается с самой первой. (АУФ)

Я понял, кем-хочу-стать-когда-вырасту, ровно 2 года назад. Кто-то скажет: "Поздновато." Да я бы и сам раньше так сказал. Но вот сейчас судить не берусь.

Свой предпринимательский путь я начал с ошибки: я даже не попытался поразмышлять "Чтобы что?". 

Это пиздец, какой риск-момент, потому что если не покопаться в причинах сразу, то со временем можно обнаружить, что купил не "ехать", а "шашечки".

// Вас когда-нибудь сокращали? У меня вот был опыт
```

**Разбор структуры:**
- Заголовок с цитатой
- Личная история
- Рефлексия про ошибку
- Яркая метафора
- Вопрос к аудитории

## Пример 2: Пост про самооценку

```
***Чтобы что?

<s>Мысль пиздец какая мудрая, хоть сейчас вешай на стену</s>

Как итог, я придумал 2 убеждения:

🟡 Ошибаться – плохо. Нужно сразу всё делать идеально.
🟡 Если я допустил ошибку, никто не должен об этом узнать.

Я завидую людям, которые могут спокойно рассказать про свои обсёры.
```

**Разбор:**
- Зачеркивание для сарказма
- Эмодзи-буллеты для списка
- Откровенность про свои проблемы

## Типичные зачины

- "Чет я думаю, что..."
- "Сегодня посреди города..."
- "В последние пару дней..."
- "Хотел сегодня рассказать про..."

## Типичные концовки

- "// Вопрос к аудитории"
- "Хер его знает, надо пробовать"
- "Короче, [вывод]"

## Длина постов

Оптимально: **800-1500 символов**
