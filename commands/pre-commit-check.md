# Команда: Проверка кода перед коммитом

## Что делает
Проверяет код на типичные проблемы перед коммитом:
1. console.log / debugger в JS/TS
2. print() для отладки в Python
3. Незакоммиченные изменения
4. Секреты в коде

## Как использовать
Пользователь пишет: `/pre-commit-check`

## Действия
1. Запусти `git status` и покажи незакоммиченные файлы
2. Проверь все изменённые файлы:
   - JS/TS: найди `console.log`, `debugger`
   - Python: найди отладочные `print()`
   - Все файлы: найди паттерны секретов (API_KEY, PASSWORD, SECRET)
3. Если найдены проблемы — выведи список с номерами строк
4. Предложи автоматически исправить (удалить console.log и т.д.)
5. После исправлений — покажи diff

## Пример вывода
```
🔍 Проверка перед коммитом:

❌ Найдены проблемы:
1. src/components/Button.tsx:15 - console.log('clicked')
2. src/utils/api.ts:42 - debugger
3. src/config.ts:8 - API_KEY = "sk-..."

Хочешь автоматически исправить? (y/n)
```
