# Команда: Очистка проекта

## Что делает
Удаляет мёртвый код, лишние файлы, неиспользуемые зависимости

## Как использовать
Пользователь пишет: `/cleanup` или `/refactor-clean`

## Действия

### 1. Проверка мёртвого кода
- Найди неиспользуемые импорты (если есть ESLint)
- Найди неиспользуемые функции
- Найди закомментированный код

### 2. Проверка лишних файлов
- Найди `.md` файлы кроме `README.md`, `CLAUDE.md`
- Найди временные файлы (`.tmp`, `.backup`, `.old`)
- Найди дубликаты кода

### 3. Проверка зависимостей (для Node.js)
```bash
npx depcheck
```

### 4. Очистка
- Предложи удалить найденные проблемы
- Покажи что будет удалено
- Спроси подтверждение перед удалением

## Пример
```
Пользователь: /cleanup

Ты:
🧹 Анализирую проект...

Найдено:
📄 3 лишних .md файла
💀 12 неиспользуемых импортов
📦 2 неиспользуемых пакета (lodash, moment)

Удалить? (y/n)
```
