# Команда: Автоформат

## Что делает
Автоматически форматирует код в нужном стиле

## Как использовать
Пользователь пишет: `/autoformat` или `/format`

## Действия

### Для JavaScript/TypeScript
1. Проверь наличие Prettier: `npx prettier --version`
2. Если есть — запусти на изменённых файлах: `npx prettier --write <файлы>`
3. Если нет Prettier — предложи установить

### Для Python
1. Проверь наличие Black: `black --version`
2. Если есть — запусти: `black <файлы>`
3. Проверь наличие isort: `isort --version`
4. Если есть — запусти: `isort <файлы>`

### Для других языков
- Go: `gofmt -w <файлы>`
- Rust: `cargo fmt`

## Пример использования
```
Пользователь: /autoformat src/components/

Ты: 
🎨 Форматирую код...
✅ Отформатировано 5 файлов через Prettier
✅ Проверены типы TypeScript - ошибок нет
```
