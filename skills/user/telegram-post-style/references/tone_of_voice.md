# Tone of Voice — Детальный анализ

## Лексика и выражения

### Характерные слова
- "чет" (вместо "что-то")
- "че-то там"
- "хер его знает"
- "пиздец какой"
- "камон"
- "факап/обсёр"
- "налажал"
- "кукуха"
- "шарю"

### Уменьшительно-ласкательные
- "книжечка"
- "листики" (про резюме)
- "пупсики" (про людей)

### Сленг и жаргон
- "корпы" (корпорации)
- "проджект" (project manager)
- "айтишник"
- "аутсорс"

## Стилистические приемы

### Самоирония
```
"Я отлично умею ошибаться в предсказаниях"
"Я не очень быстро учусь"
```

### Зачеркивания (сарказм)
```
<s>старых и красных</s> опытных и умелых
<s>смотреть рилсы</s> делать что-то понятное
```

## Структура аргументации

### Паттерн "История → Рефлексия"
1. Личная история
2. Что я чувствовал
3. Что я понял
4. Что с этим делать

## Взаимодействие с аудиторией

### Вопросы
```
// Вас когда-нибудь сокращали?
// А вы вообще заморачиваетесь?
// Что думаете?
```

## Длина и ритм

### Абзацы
- Короткие (2-4 предложения)
- Один абзац — одна мысль

### Общая длина
- 800-1500 символов
- 5-7 абзацев
- Заканчивается вопросом
