# Правила безопасности

## КРИТИЧЕСКИЕ правила

### Секреты и ключи
- ❌ НИКОГДА не хардкодить:
  - API ключи
  - Токены доступа
  - Пароли
  - Приватные ключи
  - Секреты шифрования

- ✅ ВСЕГДА использовать:
  - `.env` файлы для локальной разработки
  - Переменные окружения на продакшене
  - Менеджеры секретов (AWS Secrets Manager, etc)

### Перед каждым коммитом
Проверяй файлы на паттерны:
```bash
# Опасные паттерны
API_KEY=
SECRET=
PASSWORD=
TOKEN=
private_key
sk-...  # OpenAI ключи
ghp_... # GitHub токены
```

## Валидация данных

### Пользовательский ввод
- ✅ ВСЕГДА валидируй ввод
- ✅ Используй белый список (что разрешено)
- ❌ НЕ используй чёрный список (что запрещено)

### SQL инъекции
```typescript
// ❌ ОПАСНО
const query = `SELECT * FROM users WHERE id = ${userId}`;

// ✅ БЕЗОПАСНО
const query = 'SELECT * FROM users WHERE id = ?';
db.query(query, [userId]);
```

### XSS (Cross-Site Scripting)
```typescript
// ❌ ОПАСНО
element.innerHTML = userInput;

// ✅ БЕЗОПАСНО
element.textContent = userInput;
// или используй библиотеку для санитизации
```

## Аутентификация и авторизация

### Пароли
- ✅ Хеширование: bcrypt, argon2
- ✅ Минимум 8 символов
- ❌ НЕ храни пароли в открытом виде
- ❌ НЕ логируй пароли

### Токены
- ✅ JWT с коротким временем жизни
- ✅ Refresh tokens в безопасном хранилище
- ✅ Ротация токенов
- ❌ НЕ храни токены в localStorage (если возможно)

### Права доступа
```typescript
// ✅ Проверяй права перед действием
if (!user.isAdmin()) {
  throw new Error('Нет прав доступа');
}

// ❌ НЕ полагайся только на фронтенд
```

## Зависимости

### npm пакеты
```bash
# Проверка уязвимостей
npm audit

# Автоисправление
npm audit fix
```

### Регулярные обновления
- Обновляй зависимости минимум раз в месяц
- Читай changelogs перед обновлением
- Тестируй после обновления

## HTTPS и шифрование

### В продакшене
- ✅ ВСЕГДА используй HTTPS
- ✅ Современные TLS версии (1.2+)
- ✅ Валидные SSL сертификаты
- ❌ НЕ отключай проверку сертификатов

## Логирование

### Что логировать
- ✅ Ошибки приложения
- ✅ Попытки доступа
- ✅ Важные события

### Что НЕ логировать
- ❌ Пароли
- ❌ Токены
- ❌ Персональные данные (email, телефоны)
- ❌ Номера кредитных карт

## CORS

```typescript
// ❌ ОПАСНО
app.use(cors({ origin: '*' }));

// ✅ БЕЗОПАСНО
app.use(cors({ 
  origin: 'https://твой-домен.com',
  credentials: true 
}));
```

## Rate Limiting

```typescript
// Защита от DDoS
import rateLimit from 'express-rate-limit';

const limiter = rateLimit({
  windowMs: 15 * 60 * 1000, // 15 минут
  max: 100 // макс 100 запросов
});

app.use('/api/', limiter);
```
